# Landing Page Blueprint (Refactor Target)

## Final Page Structure (Top to Bottom)

1. Sticky Navigation
- Product, Screens, Pricing, Contact, single primary CTA.

2. Hero
- Clear value proposition, trust signal, primary CTA, secondary CTA.
- Right visual: dashboard overview screenshot.

3. Platform Overview (Merged)
- 3-4 concise value pillars.
- This replaces repeated feature grids.

4. Product Walkthrough (Slideshow + Real Screenshots)
- Left: feature title, one-liner value, short bullets.
- Right: screenshot slideshow with controls and dots.

5. Social Proof
- Testimonials + metrics in one compact section.

6. Pricing
- 3 plans, concise bullets, single CTA style.

7. Trust Strip
- Compact security and trust highlights.

8. Final CTA + Contact Form
- Strong conversion block with demo/contact.

9. Footer
- Product/company/legal links.
