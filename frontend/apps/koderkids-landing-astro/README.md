# KoderKids Landing Astro App

Static-first Astro migration target for the standalone marketing site.

## Location

- frontend/apps/koderkids-landing-astro

## Run Locally

1. Open a terminal in this folder.
2. Install dependencies:

```sh
npm install
```

3. Start dev server:

```sh
npm run dev
```

## Build and Preview

```sh
npm run build
npm run preview
```

## Current Bootstrap Coverage (Phase 1)

- Astro static output configured
- React integration enabled for islands
- Shared base layout and SEO head contract added
- Starter marketing routes in place: /, /privacy, /terms, /support
- Public assets copied from existing Vite landing app
